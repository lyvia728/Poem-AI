 🤖 AI Poem Generator

 📌 Description

The AI Poem Generator is an application that creates poems automatically based on user input. Users can enter a word, topic, or theme, and the system generates a creative poem related to that input.

This project demonstrates my skills in programming, user input handling, and creative application development.

---

 🚀 Features

* Accepts user input (word or topic)
* Generates a poem based on the input
* Displays the generated poem
* Simple and user-friendly interface
* Creative text generation

---

🛠️ Technologies Used



* Python / JavaScript
* HTML
* CSS
* AI or text-generation logic

---

🎯 Purpose of the Project

The purpose of this project is to explore creative programming and build an application that generates text automatically. It also helps improve skills in logic building and user interaction.

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/ai-poem-generator.git

2. Open the project folder.

3. Run the project:

* If Python:
  python main.py

* If Web-based:
  Open index.html in your browser.
